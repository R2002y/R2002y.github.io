1. 它是什么？

  • 基于 Web 技术开发的小型程序，可嵌入 Edge 浏览器，扩展其功能（如拦截广告、翻译网页、管理标签等），本质是遵循浏览器扩展 API 规范的代码包。

2. 它有什么用？

  • 个性化浏览器：比如自定义首页、主题样式。
  • 提升效率：比如一键保存网页内容、自动填充表单。
  • 解决特定需求：比如屏蔽特定网站、统计浏览时长等。

3. 它能和什么进行联动？

  • 能调用 Edge 浏览器自带的功能接口（如控制标签、存储数据）；
  • 可结合网页技术（如 JS 框架）或外部工具（如翻译 API）；
  • 能与云端服务、本地程序交互传递数据。
